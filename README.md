# VQA Text Encoding Models + L2 Regularization

This repository contains model variants used in the paper:

**VQA Model: Based on Different Text Encoding Models and L2 Regularization**

## Contents

- `models/`
  - `RNN.py`, `GRU.py`, `LSTM.py`
  - `Addattention_RNN.py`, `Addattention_GRU.py`, `Addattention_LSTM.py`
  - `TransformerEncoder.py`
- `paper/`
  - VQA Model Based on Different Text Encoding Models and L2 Regularization (PDF)

## Notes

- Dataset and full training pipeline are not included in this repository.
- These files focus on swapping different text encoders for VQA experiments.
- This paper was accepted by CVIV 2024 but was not published in the proceedings (camera-ready not submitted / registration not completed). No copyright transfer was signed; the authors retain the copyright.
